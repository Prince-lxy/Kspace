### 1 关系的哲学

关系的源与意义：
  - 我连接所以我存在。从心理学角度来看，人的存在可以分为三种情况，第一种是物质世界上的存在；第二种是内心世界的存在；第三种就是人际世界的存在。当我们具备了物质基础与思维后，能够决定我们如何存在就是我们的社会关系。
  - 你是一个什么样的人，取决于你处在一个什么样的社会网络中。你有什么样的品质也是基于你所处的关系网络来评判的。
  - 当我们所处的关系网变得越来越不稳固的时候，当你与网络中其他节点的联系在不断减弱的时候，你会愈发明显的感受到存在感的缺失；而当你在不断的拓宽你的关系网，提高你在关系网中的地位时，你的存在感就会逐渐的变强。在这个过程中，连接的深度比连接的广度更重要，不是看你认识多少人，而是看你能够给其他人提供多少价值。

建立关系的四个阶段：理想期、冲突期、整合期、协同创作期。
  - 理想期：因为对方的闪光点而相聚到一起的阶段。在这个阶段中每个人都在对方身上去寻找自己理想的样子，同时也会花额外的精力去扮演对方喜欢的样子，每个人都活在对方的预期之中。这个阶段大家都没有真正的走入关系之中。
  - 冲突期：当一次又一次的发现对方身上存在不符合我们预期的特点时，我们进入了冲突期。在这个阶段最常见的就是指责与自责，如果双方迟迟不能从指责与自责中走出来的话，由于精力有限，留给双方的也就就只有逃避和分离。
  - 整合期：这个阶段双方能够认识到问题其实是出自我们对对方的不切实际的幻想，源自于双方的差异。大家在这个过程中愿意努力去改变自己，开始慢慢的整合到一起。这个过程是痛苦的，也是美好的，也是属于彼此最美好最特别的回忆。
  - 协同创作期：经历了一次又一次的整合后，彼此之间产生出了前所未有的信赖。这时候彼此就能够一起去冒险，一起去创造，一起去面对生活的不确定。

关系度公式：
  - `关系度=趋近度 + 接触度 + 确认度`
    - 趋近度：指你和对方的物理距离和共享空间。趋近度是建立人际关系的基本要素，即使一言不发也能让人关注彼此，一张熟悉的面孔比陌生的更具有吸引力。但是要注意这个这个距离和空间一定要是没有威胁的，否则会激发对方的防御机制，引起逆反心理。
    - 接触度：指你和对方在一定时间内接触的次数和每次接触的时间长度。
    - 确认度：指的是你与对方关系确认的仪式。关系的确认是水到渠成的事情，但是仪式性的是与否会让你和对方的关系发生天翻地覆的改变。

关系资本可能是正资本，也可能是负资本。

搞关系的核心不是简单的讨好大人物，需要的时候可以作为一种方法，但收益和风险同样高。

大人物谁都想讨好，而且大人物往往是搞关系的优势方，所以讨好大人物非但不容易成功，而且即使讨好成功了，你也很容易成为大人物的一杆枪一颗棋子，即使因此解决了饭碗问题，哪怕是金饭碗不过也是干的讨饭的活，这样即使有了关系，也失去了自由。

没有任何一段关系可以做到百分百的绝对靠得住。就算是最极端的自己与自己的关系也不是绝对靠得住，毕竟昨天的你和今天的你并不是同一个人。单纯的追求情比金坚的关系往往很容易反受其害，越亲密的关系其实也越脆弱，越经不起挑拨。

### 2 关系的方法论

#### 2-1 是与否

|是与否|含义|
|  --  | -- |
|否|修炼核心技能时请关闭关系网|
|是|处在人的社会，自降生起就自然成为了网络中的一个节点|

#### 2-2 如何搭关系网

<趋近度> 评估趋近度()
{
	分析现存的物理距离的长短和共享空间的维度()
	分析当前的物理距离和共享空间是否会对对方造成威胁，引起对方的心理防御机制()
	趋近度 = 计算和评估目前与对方的趋近度水平()
	return 趋近度;
}

<趋近度> 调整趋近度 ( 关系性质 )
{
	当前趋近度 = 评估趋近度();

	if ( 关系性质 == 增强 ){
		if ( 当前趋近度 == 适度 ) {
			保持;
			return <当前趋近度>；
		} else if ( 当前趋近度 == 过高 ) {
			// 距离产生美
			趋近度 = 适度降低();
			return <趋近度>；
		} else {
			// 错误归因：当人自我感觉良好，确无法将这种良好的感觉归于特定原因时，
			//           对方会下意识把愉快的心情和自己周围的人与事物建立链接，
			//           如果你恰好是这个人，这个人也会对你产生好感。
			//
			// 微表情：在交流过程中留下好的姿态，比如在目光接触时主动微笑、歪头或者挑眉。
			//
			// 移情表述：人们总是理所当然的认为自己应当受到关注。所以要将谈话的焦点放在对方的感受上，
			//           让对方有一种被尊重的感觉。
			在对方心情愉快的时间与之接触();

			// 情绪与化学物质：人在运动后或者从劳累中解脱出来后大脑会释放内啡肽，即脑内吗啡，
			//                 会带来没有指向的与快感。
			约对方一起运动，并在结束后一起聊天和休息();

			趋近度 = 评估调整结果();

			return <趋近度>;
		}
	}

	if ( 关系性质 == 减弱 ) {
		if ( 当前趋近度 == 高 ) {
			// 错误归因：当人自我感觉不好，确无法将这种良好的感觉归于特定原因时，
			//           对方会下意识把糟糕的心情和自己周围的人与事物建立链接，
			//           如果你恰好是这个人，这个人也会对你产生厌恶感，觉得你是丧门星。
			//
			// 微表情：在交流过程中留下不好的姿态，比如时刻保持傲慢、无礼。
			//
			// 移情表述：人们总是理所当然的认为自己应当受到关注。所以不要将谈话的焦点放在对方的感受上，
			//           让对方有一种被忽略的感觉。
			在对方心情糟糕的时间与之接触();

			// 情绪与化学物质：人在运动后或者从劳累中解脱出来后大脑会释放内啡肽，即脑内吗啡，
			//                 会带来没有指向的与快感。
			拒绝与对方一起运动();

			趋近度 = 评估调整结果();

			return <趋近度>;
		} else {
			保持;
			return <当前趋近度>;
		}
	}
}

<接触度> 评估接触度()
{
	接触度 = 计算和评估目前与对方的接触频率和时长();
	return 目前与对方的接触度;
}

<接触度> 调整接触度 ( 关系性质 )
{
	当前接接触度 = 评估接触度();

	if ( 关系性质 == 增强 ){
		if ( 当前接触度 == 过高 || 当前接触度 == 过低 ) {
			// 好奇心圈套：好奇心是人类的一种本能，一种出于自我保护、繁殖和贪婪的本能。
			//
			// 稀缺法则：人总会被自己无法轻易得到的人或事物吸引，因此我们在发展关系的初期阶段
			//           无条件的迎合自己心目中的长期关系对象，一定程度的不迎合会让你显得更有
			//           神秘感和挑战性。
			调整到适度();

			接触度 = 评估调整结果();

			return <接触度>;
		} else {
			保持;
			return <当前接触度>;
		}
	}

	if ( 关系性质 == 减弱 ){
		// 过度暴露：高清无码更容易让人觉得索然无味。
		//
		// 稀缺法则：太容易的到的东西最容易被人忽略。
		过度接触();

		接触度 = 评估调整结果();

		return <接触度>;
	}
}

// 计算确认度
// 返回正数表示建立关系的程度；返回负数表示撇清关系的程度；返回0表示失败；返回值的绝对值越大表示程度越强，反之越弱。
<确认度> 关系确认(关系性质)
{
	// 移情表述：人们总是理所当然的认为自己应当受到关注。所以要将谈话的焦点放在对方的感受上，
	//           让对方有一种被尊重的感觉。
	//
	// 当对方开始频繁的好奇你们之间真正关系的时候，就可以明确与对方关系了。
	确认度 = 选择合适的场合明确关系();

	return 确认度;
}

<成功/失败> 建立关系(关系性质)
{
	趋近度 = 调整趋近度(关系性质);
	接触度 = 调整接触度(关系性质);
	确认度 = 关系确认(关系性质);

	关系度 = 趋近度 + 接触度 + 确认度;

	评估关系度(关系度);

	return <成功>;
}

我们要做的是合理利用关系网而不被关系网所拖累,核心是将自己置于关系网中对其他节点有利的位置：

在了解对方的过程中，除了了解对方本身的能力外，还要考虑到对方所能连接到的关系网。

在构建关系网中，不要想着别人对你有什么用，而是想着你对别人有什么用。由于大部分人的绝对能力都差不太多，所以你对别人的用处更多体现在你所处于关系网中的位置。

避免关系网中大家都彼此熟悉，因为身处这种关系网中，大家获取信息的内容基本一致，说白了有没有你不重要。

亲人、朋友关系网另做处理，这种关系网是用来让我们寄托心灵的地方，不是拿来利用的，所以在这种关系网里不要把事情做绝。

没有一个单一的关系绝对靠得住，所以你应该扩展自己的关系网，多元丰富充实的关系网才是安全感的来源。
